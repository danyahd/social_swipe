Social Swipe makes it easy to save and organize your favorite social media content. Here's how to use it:

  1. Visit a supported platform: Twitter/X, Instagram, Reddit, or YouTube
  2. Find a post you want to save
  3. Click the Save button added by Social Swipe
  4. View saved posts by clicking the extension icon

üí° How to Use

Saving Posts

  1. Browse your favorite social media platform
  2. Look for the purple gradient "Save" button on posts
  3. Click the button to save the post
  4. You'll see a confirmation message

Viewing Saved Posts

  1. Click the Social Swipe icon in your browser toolbar
  2. Browse through all your saved posts
  3. Use the search bar to find specific content
  4. Click "Open" to view the original post

Adding Tags

  1. Open the extension popup
  2. Find the post you want to tag
  3. Type a tag in the input field below the post
  4. Press Enter to add the tag
  5. Click the √ó on a tag to remove it

Exporting Your Data

  1. Click the extension icon
  2. Click "üì• Export" in the toolbar
  3. A JSON file will be downloaded to your computer
  4. Save this file as a backup

Importing Data

  1. Click the extension icon
  2. Click "üì§ Import" in the toolbar
  3. Select your backup JSON file
  4. Choose to merge with existing posts or replace them

‚ùì Troubleshooting

Save button not appearing?

  • Refresh the page (F5)
  • Make sure the extension is enabled
  • Check if the site is supported (Twitter, Instagram, Reddit, YouTube)
  • Try disabling and re-enabling the extension

Posts not saving?

  • Check your browser's storage permissions
  • Make sure you have enough storage space
  • Try restarting your browser

Can't find a saved post?

  • Use the search bar to search by keywords
  • Check if you accidentally deleted it
  • Look through all platforms (use platform emojis to filter)
üí° Pro Tip: Export your posts regularly to create backups. This way, you'll never lose your saved content even if you reinstall the extension or switch browsers!
